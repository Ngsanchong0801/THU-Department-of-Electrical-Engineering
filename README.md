# THU-Department-of-Electrical-Engineering
## 简介
作者为清华大学2022级电气工程及其自动化专业（能源互联网国际班）本科生  
本项目记录了作者在清华电机系就读本科期间的课程资料,希望能对未来的电机学子们提供一定帮助!  
(虽然自2025年起电机系就不再招收本科生, 名额都被分配至各书院)
